ADJUSTMENTS
In this menu, certain parameters of the AI can be changed. The parameters directly change the learning behavior of the machine learning. Hover over the question marks to get precise information.
Learning rate

The learning rate controls how large the changes to the model are. If the learning rate is too small, it takes a very long time to find a solution, while too large a learning rate can cause the model to reach a suboptimal solution too quickly. The learning rate can be a number between 0.0 and 1.0.

0
Momentum

The momentum controls how much the previous updates of the model affect the current update of the model. The momentum can be a number between 0.0 and 1.0.

0
Dropout rate

The dropout rate reduces overfitting (the model is too specific to the training data) by dropping out certain pixel. The dropout rate is set through a probability between 0.0 and 1.0 with which a given pixel is dropped out.

0
Loss function

The loss function measures the distance between the prediction of the neural network compared to the correct answer.

Epochs

In each epoch, the model iterates over the entire data set and gets evaluated at the end of the epoch. An epoch consists of smaller steps called batches. This slider determines how many epochs the model should run through.

1
Batch size

The batch size determines how large each step in the training and testing should be.

1
