HELP

The Software
The purpose of this software is to train a neural network for number recognition. This is accomplished by utilizing a specific neural network provided by the software and training it on a widely recognized dataset known as the MNIST dataset.
Start
When you are ready to start the training you can press the start button.
Adjustments
In the Adjustments tab, you will discover settings that can modify the behavior of the neural network training process. Furthermore, you can read about the effects the values have, by hovering over the questionmark symbols.
Diagram
The Diagram tab provides options for customizing the content and presentation of the diagrams on the left.
History
The History tab provides a detailed overview of the training progress. Within this tab,you can track the values used for each run and gain deeper insights into the training process.
Saving and loading models
the Models tab enables you to save the current model, providing the flexibility to train your neural network while having the option to revert to a specific model state when needed.
Testing the model
The Graph provides valuable insights into the training of the model. Additionally, you have the option to switch to the 'Test' window, where you can evaluate a hand-drawn number using the current model, gaining real-time feedback on its recognition capabilities.
