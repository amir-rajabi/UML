HELP

The Software
The purpose of this software is to train a neural network for number recognition. This is accomplished by utilizing a specific neural network provided by the software and training it on a widely recognized dataset known as the MNIST dataset, consisting of around 70.000 images of handdrawn numbers. During each training epoch the neural network goes through the entire data set, through smaller batches, trying to learn connections between the images and the number it represents.
Start
When you are ready to start the training you can press the start button. This will take in account all the adjustments you have made beforehand and start going through each epoch, while giving you feedback on the progress through the bar seen right of the start button.
Note that an epoch can last up to 40 seconds.
Adjustments
In the Adjustments tab, you will discover settings that can modify the behavior of the neural network training process. Furthermore, you can read about the effects the values have, by hovering over the questionmark symbol. To make new values take an effect, you can set the sliders to the desired value and observe the effect by starting a new run. During a run the same value will be taken into account even if you change the sliders while the run is ongoing.
Diagram
The Diagram tab provides options for customizing the content and presentation of the diagrams on the left. You have the option of looking into the loss and the accuracy for the testing set and training set. While accuracy describes how well the model can predict the labels of a data set, loss measures how offset the predictions were from the actual labels. There is also a difference between training and test set, so that one can measure the performance of the neural network on images that the it has not seen before.
History
The History tab provides a detailed overview of the training progress. Within this tab,you can track the values used for each run and gain deeper insights into the training process.
Saving and loading models
the Models tab enables you to save the current model, providing the flexibility to train your neural network while having the option to revert to a specific model state when needed.
Testing the model
The Graph provides valuable insights into the training of the model. Additionally, you have the option to switch to the 'Test' window, where you can evaluate a hand-drawn number using the current model, gaining real-time feedback on its recognition capabilities.
